## 🧭 What This Project Is

The **DTC Recruitment Portal** is not just an application system — it’s a **fully automated talent intelligence platform**.

It transforms traditional hiring into a **guided, data-driven, multi-stage experience** where every candidate is evaluated with precision, fairness, and speed.

---

## ✨ Key Highlights (Why this stands out)

🔥 Real-time adaptive assessments that change based on candidate responses
🧠 AI-inspired scoring logic that evaluates skill depth, not just answers
⚡ Lightning-fast UI with smooth transitions and motion design
📄 Instant onboarding letters generated like real corporate offers
📊 Admin dashboard that behaves like a mini HR control tower
🎯 Structured pipeline from application → decision → onboarding

---

## 🌀 Recruitment Journey Flow

```text
📝 Application Submitted
```

## 🧠 Core System Intelligence

### 🧪 Assessment Engine

* Evaluates candidates step-by-step
* Tracks consistency across answers
* Applies weighted scoring logic

### 📊 Decision Engine

* Converts raw scores into categories:

  * 🟢 Elite Candidate
  * 🟡 Standard Candidate
  * 🔴 Not Selected

### 📄 Offer Generator

* Auto-creates professional PDF letters
* Includes:

  * Candidate details
  * Role offer
  * Acceptance status
  * Secure signature block

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/your-org/dtc-recruitment-portal.git

# Navigate into project
cd dtc-recruitment-portal

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🌐 Environment Setup

Create a `.env` file:

```env
VITE_APP_NAME=DTC Recruitment Portal
VITE_API_URL=your_api_url_here
```
